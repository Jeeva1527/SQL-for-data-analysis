# SQL for Data Analysis

This project demonstrates basic to intermediate SQL queries used for data analysis on the **Superstore** dataset. It includes operations like total sales calculation, average sales, grouping, filtering, subqueries, and views.

## Project Objective

The main objective of this project is to analyze sales data using SQL and understand how SQL can be used for business insights and decision-making.

## Dataset Used

Dataset Name: **Superstore**

The dataset contains information such as:

- Sales
- Region
- Customer Name
- Orders
- Revenue details

## SQL Concepts Used

This project covers:

- SELECT statement
- Aggregate Functions (`SUM`, `AVG`)
- GROUP BY
- ORDER BY
- WHERE clause
- HAVING clause
- Subqueries
- Views
- LIMIT
